# Windows 98 Web Sim Beta
Windows 98 a böngésződben — teljesen működőképes asztal, Start menü, Jegyzettömb, Paint, Internet Explorer, Windows Media Player, 3D Pinball, Aknakereső és még sok klasszikus alkalmazás és játék!
## Figyelem: Ez a web szimulátor jelenleg BÉTA állapotban van, így előfordulhatnak hibák vagy hiányzó funkciók. Élvezd a nosztalgikus retro élményt közvetlenül a böngésződből, bármilyen eszközön, és oszd meg velünk a visszajelzéseidet a fejlesztéshez!
